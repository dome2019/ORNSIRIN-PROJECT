# Ornsirin Health Link

เดโมแอปบริการสุขภาพสำหรับลูกบ้านโครงการอรสิริน เชียงใหม่ — prototype สำหรับนำเสนอ (ไม่ใช่ระบบจริง)

## ฟีเจอร์ในเดโม

- **ปรึกษาหมอ** — เลือกแผนก (จิตแพทย์เป็นบริการเด่น) → เลือกแพทย์ → เล่าอาการ → เข้าคิว VDO call พร้อมคิวเดินจริงและชำระเงินยืนยันคิว
- **เจาะเลือดถึงบ้าน** — เลือกรายการตรวจ วันเวลา คิดยอดอัตโนมัติ
- **ยาต่อเนื่อง** — ยาประจำรายบุคคล + สั่งยาสามัญเพิ่ม แยกตามสมาชิกครอบครัว
- **บริการไลฟ์สไตล์** — แม่บ้าน · นวด · Ice bath · กายภาพบำบัด · Pet care เลือกสถานที่และผู้ให้บริการได้
- **ระบบนัดหมาย** — จ่ายผ่าน QR พร้อมเพย์ (จำลอง) → นัดขึ้นหน้าแรกพร้อมสถานะยืนยัน กดดูรายละเอียด/ยกเลิกได้

## วิธีเปิด

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย — ไม่ต้องติดตั้งอะไร และ**ไม่ต้องต่ออินเทอร์เน็ต**
(Tailwind, ไอคอน Lucide และฟอนต์ Noto ถูกเก็บไว้ในโฟลเดอร์ `vendor/` ครบแล้ว)
เฟรมมือถือย่อขนาดให้พอดีจออัตโนมัติ ใช้กับจอโปรเจกเตอร์ 1366×768 ได้
