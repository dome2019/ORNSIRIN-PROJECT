# Ornsirin Health Link

เดโมแอปบริการสุขภาพสำหรับลูกบ้านโครงการอรสิริน เชียงใหม่ — prototype สำหรับนำเสนอ (ไม่ใช่ระบบจริง ข้อมูลทั้งหมดเป็นข้อมูลจำลอง)

## วิธีเปิด

เปิดไฟล์ `index.html` ในเบราว์เซอร์ได้เลย — **ไฟล์เดียวจบ**: Tailwind, ไอคอน Lucide, ฟอนต์ Noto ไทย
และโลโก้ ฝังอยู่ในไฟล์ครบแล้ว ก๊อปปี้ไฟล์เดียวไปเปิดเครื่องไหนก็ได้ **ไม่ต้องต่ออินเทอร์เน็ต**
เฟรมมือถือย่อขนาดให้พอดีจออัตโนมัติ ใช้กับจอโปรเจกเตอร์ 1366×768 ได้

**ความเป็นส่วนตัว:** ก่อนล็อกอินเห็นแค่หน้าต้อนรับกลาง ๆ กับปุ่มบริการ — ชื่อ นัดหมาย ยา ผลตรวจ
และสัตว์เลี้ยงจะแสดงหลังล็อกอินเท่านั้น (ล็อกอินด้วยไอดี/รหัสอะไรก็ได้ — เป็นเดโม)

### เคล็ดลับเดโมสด

- กด F5 กลางเดโมได้ งานทุกอย่างอยู่ครบ (state เก็บใน sessionStorage) · อยากเริ่มเดโมใหม่หมดจด แค่ปิดแท็บแล้วเปิดใหม่
- **คีย์ลัดผู้พรีเซนต์ (คนดูไม่มีทางรู้):** ระหว่างอยู่หน้าคิวปรึกษาหมอ กด **ลูกศรขวา** = เร่งคิวเดินตามจังหวะเล่า, **ลูกศรซ้าย** = ถอยคิวกลับ
- ออกจากหน้าคิวไปโชว์หน้าอื่นได้เลย — หน้าแรกจะมีแบนเนอร์ "แตะเพื่อกลับเข้าคิว" คิวและสถานะจ่ายเงินไม่รีเซ็ต
- พิมพ์อะไรในช่องกรอกก็ปลอดภัย — ข้อความผู้ใช้ทุกช่องถูก escape ก่อนแสดงผล หน้าไม่มีวันแตกเพราะอักขระแปลก

## ผู้ใช้ตัวอย่าง (สลับคนที่แถบบนหลังล็อกอิน)

| คน | บทบาทในเดโม |
|---|---|
| สมหญิง (ส) | ลูกบ้าน · บ้าน 12/4 · ยาประจำ 2 ตัว · มีน้องโบ้ (โกลเด้น วัคซีนใกล้ครบกำหนด) · มีผลตรวจแล็บให้เปิดดู |
| สมชาย (ช) | ลูกบ้าน · บ้าน 45/2 · ไม่มียาประจำ — โชว์สั่งยาสามัญ/เพิ่มสัตว์เลี้ยงจากศูนย์/ผลตรวจแบบยังไม่มีข้อมูล |
| คุณยายบุญมี (ย) | ลูกบ้าน · บ้าน 88/1 · **สลับแล้วเข้าโหมดผู้สูงอายุอัตโนมัติ: ทั้งแอปขยาย ~15% + การ์ด SOS ฉุกเฉินบนหน้าแรก** · ยาประจำ 1 ตัว · สัตว์เลี้ยง 2 ตัว (น้องส้ม น้องข้าวปั้น) · มีผลตรวจแล็บ |
| กานต์ (ก) | **ลูกค้าทั่วไปนอกโครงการ** — จ่ายราคาเต็ม ไว้เทียบสิทธิ์ลูกบ้านต่อหน้าคนดู |

**สิทธิ์ลูกบ้าน Ornsirin:** ส่วนลดไลฟ์สไตล์ 15% · ฟรีค่าบริการเจาะเลือดถึงบ้าน ฿300 · ส่งยาฟรี (ลูกค้าทั่วไป ฿40)

## ฟีเจอร์ในเดโม

- **ปรึกษาหมอ** — เลือกแผนก (จิตแพทย์เป็นบริการเด่น) → ทีมแพทย์เปลี่ยนตามแผนกจริง → เลือกชิปอาการ/พิมพ์เพิ่ม → คิวเดินจริง + ชำระยืนยันคิว → **ห้อง VDO call จำลองเต็มจอ** (ตัวนับเวลา ปิดไมค์/กล้อง วางสายได้ พร้อมสรุปการปรึกษา) · คิวควบคุมได้จากคีย์ลัดผู้พรีเซนต์
- **เจาะเลือดถึงบ้าน** — เลือกรายการตรวจ วันเวลา (ตั้งต้นพรุ่งนี้ 09:00 กดจองได้ทันที) ใบเสร็จแยกรายการ
- **หน้าผลตรวจแล็บ** — ปุ่ม "ผลตรวจของฉัน" บนหน้าเจาะเลือด → ผลตรวจรายสมาชิกพร้อมชิป ปกติ/ควรติดตาม วันที่คำนวณจากวันจริง และปุ่ม "ปรึกษาหมอต่อ" — เปลี่ยนเรื่องเล่าจากแอปจองเป็น health record
- **ยาต่อเนื่อง** — ยาประจำรายบุคคล + กางสั่งยาสามัญเพิ่ม สรุปยอดแยกรายตัวก่อนจ่าย
- **กระดิ่งแจ้งเตือน** — badge บนแถบบน → รายการแจ้งเตือน (ยาใกล้หมด / ผลตรวจพร้อมดู / วัคซีนสัตว์เลี้ยง) แตะแล้วเด้งไปหน้านั้นทันที และแทรก "ใกล้ถึงคิวแล้ว" อัตโนมัติเมื่อมีคิวค้าง
- **โหมดผู้สูงอายุ + SOS** — สลับเป็นคุณยายบุญมีแล้วตัวหนังสือ/ปุ่มขยายทั้งแอปทันทีโดยไม่ต้องตั้งค่า · การ์ด SOS แดงเด่นบนหน้าแรก แตะแล้วจำลองทีมดูแลโทรกลับใน 1 นาที — จุดขายสำหรับกลุ่มหมู่บ้านผู้เกษียณ
- **บริการไลฟ์สไตล์** — แม่บ้าน · นวด · Ice bath · กายภาพบำบัด · Pet care — เลือกสถานที่ ผู้ให้บริการ และ**วัน-เวลานัด** (ตารางคิวว่างต่างกันตามพนักงาน มีช่อง "มีนัดแล้ว/ผ่านไปแล้ว")
- **Pet Care ครบวงจร** — โปรไฟล์สัตว์เลี้ยงรายสมาชิก เพิ่ม/แก้ไข/ลบได้ ชิปเตือนวัคซีนกดจองสัตวแพทย์ได้ทันที ตอนจองเลือกได้ว่านัดของน้องตัวไหน
- **การชำระเงิน** — สรุปรายการให้ตรวจก่อนจ่าย → QR พร้อมเพย์ (สแกนได้จริง ขึ้นข้อความเดโม) → จังหวะ "กำลังตรวจสอบยอดชำระ..." → เอฟเฟกต์ฉลองจ่ายสำเร็จ → นัดขึ้นหน้าแรกพร้อมรหัสจอง กดดูรายละเอียด/ยกเลิกได้
- **หน้าแรกมีชีวิต** — ทักทายตามเวลาจริง วันนัดคำนวณอัตโนมัติ แบนเนอร์คิวค้าง การ์ดแนะนำบริการ (เส้นประ) แยกจากนัดจริง (ป้ายยืนยันแล้ว)
