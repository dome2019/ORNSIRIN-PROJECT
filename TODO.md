## P1 — ทำแล้วเดโมดูดีขึ้นชัดเจน (เรียงตามผลลัพธ์ต่อแรงที่ลง)

- [x] ~~จ่ายเงินยืนยันคิวแล้วขึ้น "ชำระแล้ว"~~ `S` — **เสร็จแล้ว 17 ก.ค. 2026**
  เพิ่ม `state.queuePaid` — จ่ายแล้วการ์ดเปลี่ยนเป็นเช็คเขียว "ชำระแล้ว · คิวได้รับการยืนยัน" ไม่ทวงเงินซ้ำ และรีเซ็ตเมื่อส่งคำขอปรึกษาใหม่ (ปุ่ม VDO call ยังกดได้เสมอโดยไม่บังคับจ่ายก่อน — กันทางตันตอนเดโมสด)

- [x] ~~หน้า VDO call จำลอง~~ `M` — **เสร็จแล้ว 18 ก.ค. 2026**
  หน้าจอสนทนาเต็มจอโทนเข้ม: เชื่อมต่อ ~2 วิ พร้อมวงหายใจ (ล้อ motif หน้าจิตแพทย์) → ติดสายแล้วตัวนับเวลาเดินจริง, ชื่อ-แผนกหมอตาม `selectedDoctor`, กล้องตัวเอง (PiP) มุมขวา, ปุ่มปิดไมค์/ปิดกล้อง (toggle ได้จริง) และวางสาย → dialog "จบการปรึกษาแล้ว" สรุประยะเวลา + ล้างคิวกลับหน้าแรก · refresh กลางสายแล้วกลับมาต่อได้

- [x] ~~จองแล้วเก็บเข้ารายการนัดหมายบนหน้าแรก~~ `S` — **เสร็จแล้ว 17 ก.ค. 2026**
  ทุก flow ที่จ่ายเงิน (เจาะเลือด/ยา/ไลฟ์สไตล์) push เข้า `state.bookings` แล้วโชว์บน "นัดหมายที่กำลังจะถึง" พร้อมป้ายเช็คเขียว "ยืนยันแล้ว" + ยอดเงิน เรียงใหม่สุดบนสุด — การ์ดชวนจองเดิม (ติดป้าย "ยังไม่ยืนยัน · แตะเพื่อยืนยันนัดหมาย") จะโชว์เฉพาะตอนยังไม่มีนัดจริง
  **ต่อยอดเพิ่ม:** การ์ดนัดหมายกดเข้าไปดูรายละเอียดเต็มได้ (sheet เลื่อนขึ้น: รหัสการจอง OSR-XXXX, ป้ายสถานะ, รายการตรวจ/ยา/ผู้ให้บริการ, สถานที่, ยอดชำระ) พร้อมปุ่มยกเลิกนัดหมาย → คืนเงิน + การ์ดหายจากหน้าแรก

- [x] ~~QR จริงแทนไอคอน~~ `S` — **เสร็จแล้ว 17 ก.ค. 2026**

  คิววิ่ง 18 วิ ตามนาฬิกาไม่สนจังหวะพูด และออกจากหน้าคิวแล้วกลับเข้าไม่ได้ (ต้องพิมพ์อาการใหม่ + คิวรีเซ็ต #3)
  **แนวทาง:** แบนเนอร์ "คิวของคุณ #N" บน home เมื่อ `state.queue` มีค่า (กดกลับเข้าโดยไม่รีเซ็ต) + คีย์ลัดลูกศรซ้าย/ขวา ขยับคิวตามจังหวะเล่า

- [ ] **ซ่อนข้อมูลส่วนตัวก่อนล็อกอิน** `M` — ตอนนี้ยังไม่ล็อกอินก็เห็นชื่อ+รายการยาแล้ว (เลี่ยงชั่วคราวได้ด้วยสคริปต์เดโม: ล็อกอินเป็นคลิกแรกเสมอ)
